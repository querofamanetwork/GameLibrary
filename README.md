# GameLibrary

Game recommendation, analysis, and user profiling system using Steam, IGDB, and RAWG APIs.

A personal local project focused on analyzing custom game libraries, player behavior, recommendation systems, and metadata enrichment using APIs such as Steam, RAWG, and IGDB.

## Features Planned

- Add any game from multiple available sources, or manually fill all fields yourself;
- Rate any game based on personal enjoyment and five main technical aspects, separating personal taste from critical evaluation;
- Genre and preference profiling by analyzing all available metadata;
- Predicted score system to reveal personal patterns, inconsistencies, and hidden preferences;
- Public and user review comparison;
- Detailed metadata organization using weighted layers, prioritizing personal ratings;
- Personalized recommendation engine to suggest which games you are most likely to enjoy next, based not only on your usual preferences, but also on hidden patterns you may not notice;
- User pattern analyzer to identify behavior patterns, combinations, weighted ratings, and explain them clearly;
- User profiling system that gamifies the experience by classifying players into archetypes and exploring their unique combined traits.
